书籍

  • Julia: High Performance Programming 作者 Ivo Balbaert, Avik Sengupta, Malcolm Sherrington (697 pages; published: November 2016; ISBN: 9781787125704) 在这个学习路径中，您将学习使用一种有趣且动态的编程语言-Julia！ 本书是三位作者三本独立书籍的组合和整理。
  • Julia Programming for Operations Research: A Primer on Computing 作者 Changhyun Kwon (246 pages; published: 2016-05; ISBN: 978-1533328793) 本书旨在教授如何解决运筹学和管理科学中出现的各种优化问题。
  • Julia High Performance 作者 Avik Sengupta (120 pages; published: 2016-05; ISBN: 9781785880919) 这是一本关于Julia程序性能优化的书，展示了如何设计和编写完全实现语言及其库的潜在速度的Julia代码。
  • Mastering Julia 作者 Malcolm Sherrington - 由 Packt 出版 (410 pages; published: 2015-07; ISBN: 9781783553310)
  • Getting Started with Julia Programming 作者 Ivo Balbaert - 由 Packt 出版 (214 pages; published: 2015-02-28; ISBN: 9781783284795)
  • Seven More Languages in Seven Weeks 作者 Bruce Tate, Fred Daoud, Jack Moffit and Ian Dees - 出版方: The Pragmatic Programmers (350 pages; published: 2014-11-15; ISBN: 978-1-94122-215-7) 本书包含Julia教程章节（由Jack Moffitt和Bruce Tate编写），面向Julia新手的程序员，非常好，有很好的例子和练习。
  • Julia for Data Science 作者 Anshul Joshi (348 pages; published: 2016-09; ISBN: 9781785289699) 从头开始探索数据科学世界，Julia如风，常伴吾身。
  • Julia for Data Science 作者 Zacharias Voulgaris PhD (415 pages; published: 2016-09-01; ISBN: 9781634621304). 通过Julia编程生态系统掌握数据科学的基本知识（不必懂其他语言也可），并附带各种有趣的示例和练习。
  • Julia Cookbook 作者 Jalem Raj Rohit - 出版方： Packt Publishing (172 pages; published: 2016-09; ISBN: 9781785882012)
  • Julia Solutions 作者 Jalem Raj Rohit - 为Julia程序员学习数据科学的综合指南 - 由 Packt 出版 (2 hours and 52 minutes long; published: January 31, 2017; ISBN: 9781787283299)
  • Getting Started with Julia 作者 Erik Engheim - 为了高性能技术计算学习新语言Julia - 由 Packt 出版 (9 hours and 50 minutes long; published: March 31, 2017; ISBN: 9781786462978)
  • Learning Julia 作者 Anshul Joshi, Rahul Lakhanpal (316 pages; published: November 2017; ISBN: 9781785883279). 本书向您展示了如何编写有效的函数，减少代码冗余以及改进代码重用。对于刚开始使用Julia的新程序员来说，探索其广泛且不断增长的软件包生态系统以及希望将Julia添加到他们的技能组合中的经验丰富的开发人员/统计人员/数据科学家将会很有帮助。
  • Hands-On Computer Vision with Julia 作者 Dmitrijs Cudihins (202 pages; published: June 2018; ISBN: 9781788998796). 探索Julia中支持图像处理的各种软件包，并构建用于视频处理和对象跟踪的神经网络。
  • Iterative Solution of Symmetric Quasi-Definite Linear Systems 作者 Dominique Orban 和 Mario Arioli. 本书面向研究人员和高级研究生，包括计算优化，计算流体动力学，计算线性代数，数据同化以及几乎任何发生鞍点系统的计算领域。 Krylov.jl 是本书附带的Julia软件包。Ebook 用户电子书。
